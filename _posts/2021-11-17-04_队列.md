## 队列
**这是一个很重要的知识点,可以用来解一些复杂的算法.队列一般被用作算法优化.**

这里有一个问题,加密一个数.
```c
#include <stdio.h>

struct queue
{
	int data[100];
	int head;
	int tail;
};

int main()
{
	struct queue q;
	//初始化队列
	q.head = 1; //第一个元素
	q.tail = 1; //第二个元素

	for (int i = 1; i <= 9; ++i)
	{
		scanf("%d", &q.data[q.tail]);
		q.tail++;
	}

	while(q.head < q.tail)
	{
		printf("%d ", q.data[q.head]);
		q.head++;

		q.data[q.tail] = q.data[q.head];
		q.tail++;

		q.head++;
	}

	getchar(); getchar();
	return 0;
}
```
